## 技术栈

> Vite + TS + Vue3.x + Electron + Eslint + Less + NodeJs

## 安装依赖

```Basic
npm i
```

## 运行项目

```Basic
npm run dev:electron
```

## 打包项目

```Basic
npm run build
```
