## 技术栈

> Vite + TS + Vue3.x + Electron + Eslint + Less + NodeJs

## 安装依赖

```Basic
npm i
```

## 运行项目

```Basic
npm start
```

## 打包项目

```Basic
npm run build
```

## 功能

- 激活系统
- 锁屏
- 关机
- 重启
- 定时关机
- 取消定时关机
- 设置磁盘图标
- 重置磁盘图标
- 屏幕显示
